Section 06 — 신규 추가

마스코트 전략

🧚 싱가폴꿀팁 "싱꿀이"가 증명한 것
마스코트 하나가 플랫폼 전체의 인격을 만듭니다. 유저들이 "사이트"가 아닌 "친구"에게 가는 느낌을 주고, 재방문율과 브랜드 기억도를 극적으로 높입니다. SORI도 반드시 필요합니다.

SORI 마스코트 컨셉 후보

🐦
작은 새 "소리"
싱가포르 하늘을 자유롭게 나는 작은 새. 싱가포르의 다양한 동네를 탐험하는 여행자 이미지. 국적 초월, 친근감 높음.
🧑‍🚀
한인 청년 "소리"
MRT를 타고 싱가포르를 돌아다니는 한인 20대 캐릭터. 사용자와 동일시. 2030 타겟에 최적.
🦁
한-싱 혼합 캐릭터
싱가포르 국장(사자)에서 영감을 받은 캐릭터에 한복 요소 가미. 두 문화 융합 상징. 독창성 높음.
💡 실행 팁: AI 이미지 생성(Midjourney, DALL-E) 으로 3가지 시안 제작 → 커뮤니티 투표로 최종 결정. 유저가 직접 선택한 마스코트는 애착이 2배. 비용: 거의 0원.

마스코트 활용 지점

  1. 빈 피드 화면 "아직 글이 없어요! 첫 글을 써봐요 🐦"
  2. 로딩 화면 마스코트가 MRT 타는 애니메이션
  3. 에러 페이지 "길을 잃었어요! 홈으로 돌아가볼까요?"
  4. Daily 뉴스 아바타 뉴스 큐레이터 역할로 "소리가 오늘의 뉴스를 전합니다"
  5. 인스타그램 계정 마스코트 중심의 SNS 콘텐츠 — 팔로워 확보 채널
Section 07 — 신규 추가

콘텐츠 전략

싱가폴꿀팁 분석에서 발견한 핵심: 기술보다 콘텐츠 리듬이 MAU를 만든다. 사람들이 매일, 매주 돌아올 이유를 설계합니다.

📅 콘텐츠 리듬 설계

주기콘텐츠담당목적
매일Daily SG 뉴스 (5개)AI 번역 + 운영자 편집직접 방문 습관 형성
매일속마음 방 자유 게시유저 UGC감성 연결, 체류시간↑
주 2~3회맛집 리뷰 (운영자 큐레이션)운영자콘텐츠 품질 보장
주 1회위클리 뉴스레터AI 보조 + 운영자이탈 사용자 재유입
수시핫딜 정보운영자 + 유저검색 유입 + 재방문
월 1~2회SORI 인증 챌린지 이벤트운영자 기획바이럴 + UGC 폭발
🤖 AI 자동화로 1인도 가능한 이유
Daily 뉴스: CNA/Straits Times RSS → GPT API 번역 → 자동 게시. 운영자 검토는 5분. 위클리 뉴스레터: 주간 인기글 자동 집계 → AI 초안 → 운영자 편집. 총 소요시간 1시간 이하. 싱가폴꿀팁이 수동으로 하는 걸 SORI는 반자동화.

📣 오픈 전 씨드 콘텐츠 계획

  1. 생활 가이드 50개 직접 작성 "싱가포르 은행 계좌 개설 A-Z", "EP 비자 갱신 완전 정복" 등 검색 수요 높은 롱테일. SEO 기반 확보.
  2. 맛집 리뷰 30개 한인 선호 맛집 직접 방문 또는 기존 리뷰 재구성. 업소록 초기 채움.
  3. 핫딜 10개 사전 적재 FairPrice, 주요 항공사 프로모션 정보.
  4. 업소 50개 무료 사전 등록 한인 업소 직접 연락해 무료 등재 완료 후 런칭.
Section 08

UX/UI 방향성

🎨 디자인 원칙

  1. 모바일 퍼스트, 앱처럼 느껴지는 웹 전체 시나리오를 스마트폰 기준으로. Bottom navigation. 싱가폴꿀팁보다 확연히 나은 모바일 UX.
  2. 캐주얼함 + 신뢰의 공존 감성 게시판(속마음방)은 친근하게, 비자·세금 정보는 신뢰감 있게. 섹션별 톤 분리.
  3. 신뢰 설계 (Trust Design) 모든 정보 카드에 마지막 업데이트 일시 표시. 오래된 정보는 자동 "업데이트 필요" 배지.
  4. 정보 밀도 최적화 카드·타일·리스트를 상황에 맞게 전환. 스크롤 1회로 원하는 정보 도달.
💡 킬러 UX: 홈 첫 화면의 "지금 열려있는 한식당 47곳 →" 버튼. 현재 시간 기준 영업 중인 한인 업소 즉시 필터. imweb으로는 절대 구현 불가.
