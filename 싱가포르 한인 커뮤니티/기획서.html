💡 네이밍: 왜 SORI인가?
소리(所里) = '마을이 모이는 곳' + 소리(sound) = 목소리가 있는 커뮤니티. 한국촌의 '촌(村)' 개념을 현대적으로 재해석. 영문 도메인 sori.sg 형태로 싱가포르 현지감 강화.

한국촌은 25년 역사를 지닌 싱가포르 최대 한인 사이트지만, 다음과 같은 구조적 문제를 안고 있습니다.

🔍 핵심 문제 3가지

📱
모바일 퍼스트 미달
실제 싱가포르 거주 한인은 스마트폰 중심 생활. 앱은 2020년 이후 방치, 웹은 데스크탑 중심 구조.
💬
실시간 커뮤니티 부재
카카오톡 오픈채팅방에 커뮤니티 수요를 완전히 빼앗김. 정보가 채팅방 안에 갇혀 검색 불가.
💰
단일 수익 구조
배너 광고 의존 구조. 월 트래픽 한계로 수익 천장이 낮고, 신규 수익원 개발이 없었음.
⚠️ 기회의 창
싱가포르 한인 수는 약 3~4만 명으로 추정 (외교부 2023). 현재 한국촌 실제 싱가포르 방문자는 월 2,600명 수준 — 즉 타겟 모수의 10% 이하만 커버. 나머지 90%는 카카오톡과 인스타그램으로 분산 중이며, 이 수요를 하나로 묶는 플랫폼이 존재하지 않는다.

🥊 경쟁 구도

플랫폼 강점 약점 위협 수준

카카오오픈채팅 실시간성, 이미 사용자 집중 정보 검색 불가, 아카이빙 없음 최대 위협
네이버 카페 콘텐츠 검색, 한국 인지도 싱가포르 현지화 미흡, 느린 UX 중간 위협

Section 04

핵심 기능 설계

📌 MVP 원칙
"모든 것을 한번에" 함정 금지. Phase 1은 3가지 핵심 기능만. 작게 시작해서 사용자 피드백으로 확장.

🚀 Phase 1 — MVP (출시 기준 필수)

🔍
정보 허브 (검색 중심)
생활정보 Q&A, 태그 기반 필터링, 카테고리 구조화. 카카오 채팅방의 정보를 검색 가능한 형태로 전환하는 것이 핵심 가치.
🏪
업소록 2.0
한인 업소 지도 기반 탐색, 리뷰·평점, 예약 링크 연동. 구글 맵 API 활용, 운영시간·메뉴·가격 표시.
💼
구인구직 보드
싱가포르 한인 특화 채용 공고. EP/S-Pass/PR 비자 유형별 필터, 한국어 가능 포지션 우선 표시.

🌱 Phase 2 — 커뮤니티 확장

🏘️
동네 커뮤니티
지역별(부오나비스타·선텍·주롱 등) 게시판. 벼룩시장, 카풀, 물건 나눔 등 하이퍼로컬 기능.
📅
이벤트 캘린더
한인 행사, 모임, 공연, 한인회 소식을 한눈에. 이벤트 등록 및 참가 신청 (수익화 가능).
🏠
부동산 게시판
한인 선호 지역 콘도·HDB 매물. 에이전트 연결, 임대/매매 구분. 지도 기반 탐색.

💎 Phase 3 — 수익화 심화

🎟️
티켓 & 예약 커머스
한인 투어, 공연, 모임 티켓 판매. 예약 수수료 5~10% 수취. 수익화 핵심 엔진.
📧
위클리 뉴스레터
싱가포르 한인 필수 정보 큐레이션. 광고 스폰서십 삽입. 구독자 확보 시 고단가 광고 가능.
⭐
프리미엄 멤버십
월 SGD 5~10 구독. 광고 없는 경험, 전문가 Q&A, 독점 할인 쿠폰 등 혜택 제공.
Section 05

UX/UI 방향성

🎨 디자인 원칙 4가지

  1. 모바일 퍼스트, 앱처럼 느껴지는 웹 전체 사용 시나리오를 스마트폰 기준으로 설계. 웹도 앱과 동일한 UX 제공. Bottom navigation 구조.
  2. 정보 밀도 최적화 복잡한 게시판 나열 대신, 카드·타일·리스트를 상황에 맞게 전환. 스크롤 1회로 원하는 정보 도달.
  3. 한국어 + 싱가포르 감성의 조화 한국 앱 트렌드(당근마켓·토스 스타일 친근함)에 싱가포르 컬러(현지 감성) 가미. 과한 영어 혼용 지양.
  4. 신뢰 설계 (Trust Design) 작성자 정보, 마지막 업데이트 일시, 조회수, 도움돼요 반응을 모든 정보 카드에 표시. 오래된 정보는 자동으로 "업데이트 필요" 배지 표시.

🗂️ 핵심 화면 구성

화면핵심 요소차별화 포인트
홈 피드형 최신글 + 빠른 접근 단축버튼 + 지역 날씨/환율 매일 달라지는 생활 정보 위젯 (환율·MRT 혼잡도)
탐색 카테고리 그리드 + 지도 토글 + 검색 지도와 리스트를 즉시 전환 가능
커뮤니티 게시판 + Q&A + 실시간 인기글 태그 기반 구독 (내가 관심있는 태그만 보기)
업소록 지도 + 필터 + 리뷰 + 운영시간 현재 영업 중인 곳 즉시 필터, 한국어 메뉴판 사진
MY 내 글·댓글·즐겨찾기·알림 비자 만료일 알림, 관심 업소 가격 변동 알림
💡 킬러 UX 아이디어: "지금 열려있는 한식당" 버튼 — 현재 시간 기준 영업 중인 한인 업소만 즉시 필터링. 싱가포르 거주자가 가장 자주 필요로 하는 기능을 홈 첫 화면에 배치.

🌱 콜드 스타트 문제 해결

커뮤니티 플랫폼의 최대 난관은 "닭이 먼저냐 달걀이 먼저냐" 문제. 사용자가 없으면 콘텐츠 없고, 콘텐츠 없으면 사용자 없음. 다음 순서로 해결합니다.

  1. 씨드 콘텐츠 선제 구축 오픈 전 3개월간 가이드 50개 이상 직접 작성. "싱가포르 은행 계좌 개설 방법", "EP 비자 갱신 A-Z" 등 검색 수요 높은 롱테일 콘텐츠. SEO 효과 = 오픈 즉시 유기 트래픽 확보.
  2. 업소 파트너십 선확보 오픈 전 한인 업소 50개 이상 무료 등재 완료. 업소가 자신의 채널에 소개하면 자연스럽게 바이럴.
  3. 인플루언서 오픈 파트너 싱가포르 관련 한국 유튜버·블로거 3~5명과 오픈 파트너 계약. 컨텐츠 협업 + 플랫폼 소개.
  4. 카카오 오픈채팅방 연계 주요 싱가포르 한인 채팅방에 유용한 정보 링크 공유. 채팅방을 대체하는 것이 아닌 보완 포지셔닝.

✅ 결정 사항 — 기획서 대비 변경된 점
벼룩시장·부동산을 커뮤니티 카테고리에서 분리해 별도 페이지로 승격(좌측 사이드바 메인 메뉴). 콘텐츠 성격이 게시글이 아닌 거래·매물 중심이라 UX 분리가 자연스러움. 사이드바 메뉴 순서도 한인 사용 빈도에 맞춰 재배치 — 홈 → 뉴스 → 한인업소록 → 부동산 → 벼룩시장 → 구인구직 → 커뮤니티.

2일차 추가: 한인업소록도 등록 폼·플로팅 버튼·"내 업소" 배지를 갖춰, 다른 카테고리와 동일한 UX 통일. 모바일에서는 햄버거 메뉴로 전체 메뉴 접근 + BottomNav 5개 탭 + 우측 상단 검색/알림/MY 빠른 접근 — 3가지 진입로 확보.
⚠️ 현 시점 한계 (여전히 유효)
모든 사용자 데이터(글·댓글·좋아요·저장·프로필 등)가 브라우저 localStorage 단위로만 저장됩니다. 즉, 다른 사람이 보는 화면에는 본인이 쓴 글이 안 보이며, 휴대폰에서 쓴 글이 데스크탑에서도 안 보입니다. 이는 의도된 MVP 단계의 제약이며, Supabase 연동 시 즉시 해결됩니다.
